# deeepseekAVS
1) Прописать pip install unsloth, будут установлены все нужные инструменты, по крайней мере должны, возможны конфликты
2) Установить  датасет KVP10k, проще всего перенести папку или zip архив из моего окружения, так уже уменьшенные изображения
3) Установить модель deepseek ocr, лучше сразу вторую deepseek ocr 2, или просто взять первую из моего окружения, второй вариант проще так, КАК проблемы со скоростью и доступом
4) Запустить скрипт train.py, он должен быть в той же директории, что и папка с набором данных и папка с моделью
